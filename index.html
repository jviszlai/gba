Welcome!

This website hosts my current final project for Phys-4147 and is running a gameboy emulator. The controls are as follows (in GBA -> Keyboard pairs): A -> Z, B -> X, L -> A, R -> S, SELECT -> Backspace, START -> Enter, D-Pad: Arrow Keys. You can modify the mappings using the settings menu (click on the key in the list, then tap the key you want to map it to).
